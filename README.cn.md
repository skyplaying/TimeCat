## 项目简介

中文 | [English](./README.md)

TimeCat 是一个完全开源的网页录制工具，生成的文件不是真正的视频，但可以像真正的视频一样进行播放，完美的还原用户在浏览器中的操作行为

[🖥 Demo](https://timecatjs.com/demo)
